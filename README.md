# hdminer

A miner for HDPOOL.COM, like [blagominer](https://github.com/JohnnyFFM/blagominer) and [scavenger](https://github.com/PoC-Consortium/scavenger).

## Build

### Dependencies
- go1.11.x
- gcc (8.0 or higher)

```sh
sh build.sh
```
